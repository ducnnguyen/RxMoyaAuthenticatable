## Requirements

| Target            | Version  |
|-------------------|----------|
| iOS               |  => 11.0 |
| Swift             |  => 4.0  |
| Moya/RxSwift      |  => 11.0 |

## Installation

RxMoyaAuthenticatable is available through [CocoaPods](https://cocoapods.org).
To installit, simply add the following line to your Podfile:

```ruby
pod 'RxMoyaAuthenticatable'
```
